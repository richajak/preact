# preact

Smallest Preact (React Compatible) environment

Step 1 - Get this template
`npx degit richajak/vite-project my-project`

Step 2 - Install dependency
`cd my-project`
npm install`

Step 3 - Run VITE
`vite`

That's all.

Note: 

Step 0 - Only once, install VITE as global
`npm install -g vite`
